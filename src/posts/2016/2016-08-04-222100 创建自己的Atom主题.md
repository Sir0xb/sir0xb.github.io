### 3.提交并发布

#### ==> 打 git tag

``` bash
git tag v0.2.0
```

#### ==> 提交 tag 信息

``` bash
git push --tags
```

#### ==> 发布	apm 皮肤

``` bash
apm publish --tag v0.1.0
```

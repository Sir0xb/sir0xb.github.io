到 joomla1.7 安装目录找到 `configuration.php` 文件记事本打开将

``` php
...
public $force_ssl = '2';
...
```

修改成

``` php
...
public $force_ssl = '0';
...
```

访问后台，重新设置，就跟什么都没发生一样。

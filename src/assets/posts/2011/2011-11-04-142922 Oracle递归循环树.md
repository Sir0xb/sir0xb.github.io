#### #.从一个节点搜索所有子节点

``` sql
select *                                -------可以指点欲得到的键值
from table_name                         -------表名
where 1 = 1                             -------可以添加条件限定范围
start with upper_id = 56                -------起始条件，给一个节点ID
connect by prior id = upper_id          -------通过上节点ID进行循环
```


#### #.从一个节点出发搜索根节点

``` sql
select *                                -------可以指点欲得到的键值
from table_name                         -------表名
where 1 = 1                             -------可以添加条件限定范围
start with id = 56                      -------起始条件，给一个节点ID
connect by prior upper_id = id          -------通过上节点ID进行循环
```

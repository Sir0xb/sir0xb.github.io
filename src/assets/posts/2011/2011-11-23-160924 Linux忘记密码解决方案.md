..............硬件环境页面................

#### 1. 等硬件环境页面一过就按 “E” 键。

#### 2.上下键选择 Kernel 启动方式，再次按 “E” 键。

#### 3.在出现的命令行后面加 “原有命令行 + “空格” + “single””，回车。

#### 4.在 Kernel 启动方式下，按 “B” 键启动。

..............进入单用户模式.............

<!--more-->

#### 5.在命令行中输入：passwd root

并根据提示重新输入密码。

如果账号也忘记了，可以 Cat .. /etc/passwd 文件，查看账号信息。

#### 7.输入 reboot 命令，重新启动。

..............正常模式登陆系统..............

#### 8.正常登陆并使用 root + 设置的 passwd 即可进入系统。

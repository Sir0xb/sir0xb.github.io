Mac 设置应用快捷键

Posted by Sir0xb on 2015-05-16 00:52:16 +0800

老早就想好好整理整理关于 Mac 的文章，一直没有动手。正好有了自己的网站，也算是一个契机吧，分享一下自己学习到的好功能。用 Mac 过程中，多次验证了一个真理 当你觉得 Mac 的某个功能不好用的时候，其实人家早就解决了这个问题，只是你不知道而已

遇到的问题：

之前 Chrome 切换 Tab 的快捷方式是 command + shift + 方向键 。某一次突然发现键位不好使了，变成了 command + option + 方向键 。

 

... ... 折腾之后 ... ...

 

解决方法：

先看好要设置的菜单名称

* 例如：任务管理器 *

打开系统偏好设置 > 键盘 > 快捷键

选择右侧 应用程序快捷键 之后，点击 +

* 应用程序 ：选择 Chrome * * 菜单标题 ：任务管理器 * * 键盘快捷键：｀shift + command + R｀ *

再看 Chrome 菜单的变化

 

延伸：

  • 收藏夹里的网页也可以设置快捷方式
  • 只要是菜单栏里的，系统菜单也可以设置 我把系统偏好设置成了 shift + command + , ---> 需要注意，这个设置的时候，字符串要填写 系统偏好设置...
  • 其他应用程序也可以用同样方式设置
